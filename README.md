vagrant-hls-streaming(with docker-compose)
====

vagrantで簡単にhls配信できる環境を構築する。

## Requirement
* Virtual Box
* docker
* docker-compose

## Usage

## Install
$ vagrant plugin install vagrant-vbguest
$ vagrant up
